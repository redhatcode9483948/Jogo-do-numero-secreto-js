# Jogo-do-numero-secreto-js

Nesse projeto simples, estou praticando a lógica de programação com Alura. A ideia é criar um jogo simples, onde você tem que adivinhar o número secreto. O algoritmo fala para o usuário se o chute/palpite está certo ou não e ainda exibe o número no console. Veja:

É maior ou menor que o número que o usuário tentar adivinhar.
Se for menor (<), errou!
Se for maior (>) errou!
Se for igual (==), ganhou!
